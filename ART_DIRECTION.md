# 观天模式艺术方向

观天模式不是室内道观，也不是古风工具页。它是在天地之间起卦：人坐在户外木案前，视野低而近，能看到宽袖、小臂、手、木案和蓍草；远处是草地、山影、水声和风。

## 视觉规则

- 场景基底是户外草地，低饱和、留白，不做繁复装饰。
- 远处有简化山影，形体安静，颜色偏灰蓝。
- 一侧有溪流或水面，轻微波动即可，不抢桌面和蓍草。
- 可以有薄雾，但必须是自然空气感，不是玄幻特效。
- 天光以清晨或傍晚为准，背景不宜过亮，桌面和蓍草必须清晰可读。
- 人物只出现第一视角可见部分：宽袖、小臂、手。不露脸，不出现完整身体。
- 色彩以灰蓝、枯白、木色、草色为主，少量暖光只用于桌面可读性。
- UI 克制，像仪式中的旁注，不像游戏 HUD。
- 动态只做低成本自然运动：风吹草动、水面波纹、烛火或环境光轻微变化。
- 鸟鸣、风声、水流声可以存在，但应轻、远、稀疏。

## 禁止

- 仙侠手游风。
- 红金古风边框。
- 太极图乱飞。
- 发光按钮。
- 玄幻粒子。
- 过度中国风模板。
- 浮夸光柱、法阵、爆闪、角色立绘。
- 把 UI 做成满屏操作面板。

## 当前 MVP 取舍

- 使用几何体搭建草地、木案、远山、溪流与薄雾。
- 不引入外部素材和大型模型。
- 音频管理器先提供接口和注释占位，等待后续素材。
- 优先保证 49 根蓍草和十八变时间轴不被环境遮挡。
